# 🍊 JeolgamAI (절감이)

### 클라우드 비용을 줄이고, 팀 생산성을 높이는 FinOps 플랫폼

JeolgamAI는 클라우드 비용, 인프라 리스크, 최적화 권고, 실행 리포트를 한 번에 제공하는 서비스입니다.  
AWS·Kubernetes·Prometheus 데이터를 기반으로 낭비 리소스를 분석하고, 운영팀이 바로 실행할 수 있는 최적화 가이드를 제공합니다.

## 📚 배운 점

 클라우드 비용 데이터와 인프라 메트릭을 함께 분석하는 FinOps 관점을 익혔습니다.
 
 Kubernetes 환경에서 리소스 효율화가 비용 절감과 직접적으로 연결된다는 점을 경험했습니다.
 
 Jenkins, Docker, Kubernetes, ArgoCD를 활용해 CI/CD와 GitOps 배포 흐름을 구축했습니다.
 
 Prometheus와 Grafana를 활용해 운영 지표를 시각화하고 모니터링하는 경험을 쌓았습니다.
 
 프론트엔드 화면 구현뿐 아니라 인프라 운영 흐름까지 고려하며 서비스 전체 구조를 이해했습니다.
