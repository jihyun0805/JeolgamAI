# 🍊 JeolgamAI (절감이)

### 클라우드 비용을 줄이고, 팀 생산성을 높이는 FinOps 플랫폼

JeolgamAI는 클라우드 비용, 인프라 리스크, 최적화 권고, 실행 리포트를 한 번에 제공하는 서비스입니다.  
AWS·Kubernetes·Prometheus 데이터를 기반으로 낭비 리소스를 분석하고, 운영팀이 바로 실행할 수 있는 최적화 가이드를 제공합니다.

# 👨‍💻 담당 역할

### CI/CD 및 인프라 구축
- 가상화 환경을 위한 Docker 이미지 빌드 및 운영 서버 배포 자동화
- Prometheus, Grafana 기반 인프라 메트릭 수집 및 모니터링 대시보드 환경 구성

### Frontend 개발
- AWS 및 클라우드 비용 분석 대시보드 화면 구현
- Prometheus 메트릭 연동을 통한 클라우드 모니터링 화면 구현
- AI 비용 최적화 권고 가이드 및 요약 화면 구현
- 운영 효율 검토를 위한 통합 리포트 결과 화면 구현
- REST API 연동 및 다양한 차트 라이브러리를 활용한 데이터 시각화

- Kubernetes 환경 내 방치되거나 과할당된 낭비 리소스 약 27% 식별
- 리소스 최적화 권고 가이드 적용 시 전체 인프라 비용의 약 8% 절감 효과 확인
- 가시성 높은 클라우드 비용 데이터 제공을 통해 정량적 기반의 의사결정 지원
- Prometheus + Grafana 기반 통합 관제 시스템 구축으로 실시간 인프라 상태 파악 확보
- 중앙 집중형 로그 및 트레이싱 환경 설정을 통한 장애 탐지 및 원인 분석 효율성 증대

# 📚 배운 점

- **FinOps 관점의 정립:** 단순 모니터링을 넘어 클라우드 비용 데이터와 실시간 인프라 메트릭을 유기적으로 분석하는 역량을 익혔습니다.
- **클라우드 자원 효율화 경험:** Kubernetes 환경에서 유휴 자원을 찾아내고 효율화하는 과정이 비즈니스 비용 절감으로 직결되는 흐름을 체득했습니다.
- **GitOps 배포 환경 이해:** Jenkins, Docker, Kubernetes, ArgoCD를 연계하여 지속적 통합 및 자동 배포를 수행하는 인프라 아키텍처를 경험했습니다.
- **데이터 시각화 역량 성장:** 인프라에서 수집된 방대한 모니터링 지표를 프론트엔드 화면에 직관적으로 시각화하여 사용성을 극대화했습니다.
